# SendMoneyOnline
Currency Conversion and Money Transfer Web App

## Web Site Purpose

The project name is Jellyfish and the website's primary objective is to enable users to perform currency 
conversions, transfer money, and access information on currency exchange rates for guests and logged 
in user as personal or business accounts. Upon loading the website, users are presented with the main 
page featuring a navigation bar. The navigation bar includes the site logo and two options: "Personal" 
and "Business," representing different account types tailored to distinct user needs.

## ➢ The "Personal" account option provides the following functionalities:
 - Currency Conversion: Users can convert currencies based on current exchange rates.
 - View Rates: Users can access information on currency exchange rates.
 - Money Transfers: Users can transfer money from their account to other user accounts.
 - View Money Transfers: Users can review their money transfer history.
 
## ➢ Business Account
 The "Business" account option offers additional functionalities for companies, organizations, and 
individuals with advanced foreign exchange needs:
 - Currency Conversion: Users can perform currency exchange transactions.
 - View Rates: Users can access currency exchange rate information.
 - Money Transfers: Users can initiate money transfers to other user accounts.
 - View Transfers: Users can view details of their money transfers, including transaction history 
and status.
 - Reports and Analytics: Users have access to features that generate reports and analytics related 
to their financial transactions.

## ➢ User Guide
 The website includes a user guide section to assist new users in navigating and utilizing its features 
effectively. The user guide provides step-by-step instructions on how to perform actions such as 
currency conversion, money transfers, and accessing helpful information.

## ➢ Design and Interface
 The website features a visually appealing design centered around the Jellyfish character. The 
navigation bar is prominently displayed, offering easy access to various functionalities. A dynamic 
table and navigation tabs facilitate seamless interaction with the "Converter" and "Send Money" 
functionalities. The website also incorporates a "Guide" section to provide information similar to 
"How to send money."

## ➢ Account Management
 The website enables users to create accounts by clicking on the login and register buttons in the 
navigation bar. Account creation is straightforward, allowing users to manage their personal and 
business information.

## ➢ Dark Mode/Light Mode
 To enhance the user experience, the website includes a toggle switch in the navigation bar, allowing 
users to switch between dark mode and light mode based on their visual preferences

Here is the website's functionality for the three actors:
## 1. Guest User:
 - Exchange Currency: The guest user can perform currency exchange transactions, converting one 
currency to another.
 - View Rates: The guest user can view currency exchange rates, allowing them to see the current 
conversion rates for different currencies.
## 2. Logged User (Personal):
 - Exchange Currency: Similar to the guest user, the logged user (personal) can perform currency 
exchange transactions.
 - View Rates: The logged user (personal) can view currency exchange rates.
 - Make Money Transfers: The logged user (personal) has the ability to transfer money from their 
account to another user's account.
 - View Money Transfers: The logged user (personal) can view their own money transfer history, 
allowing them to see past transactions they have made.
## 3. Logged User (Business) which is underdevelopment functionality is supposed to perform the 
following:
 - Exchange Currency: The logged user (business) can perform currency exchange transactions.
 - View Rates: The logged user (business) can view currency exchange rates.
 - Make Money Transfers: Similar to the logged user (personal), the logged user (business) can 
transfer money to other user accounts.
 - View Transfers: The logged user (business) can view the details of their money transfers, including 
the transaction history and status.
 - See Reports and Analytics: The logged user (business) has access to additional features, such as 
generating reports and analytics related to their financial transactions. This functionality allows them 
to gain insights into their financial activities and performance.
